# Company Seal Manager

A professional seal management application for adding authentic company and director seals to invoices and documents.

## Features

✅ **Seal Types**
- Company Seal (For Mcharv Techlabs Pvt. Ltd.)
- Director Seal (Director - Shwita)

✅ **Customization Options**
- Position Control (X and Y offset)
- Scale Adjustment (50% - 200%)
- Rotation Control (0° - 360°)
- Opacity Control (0% - 100%)

✅ **Stamp Management**
- Real-time preview
- Draggable stamp placement
- Export stamp configurations
- Apply stamps to documents

## Quick Start

### Prerequisites
- Node.js (optional, for backend service)
- Modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

1. **Navigate to the folder:**
   ```bash
   cd CompanyStampManager
   ```

2. **Open in browser:**
   Simply open `index.html` in your web browser.

### Usage

1. **Select a Stamp Type**
   - Choose between Company Stamp or Director Stamp

2. **Customize Settings**
   - Adjust horizontal and vertical offset
   - Set scale percentage
   - Rotate to desired angle
   - Control opacity level

3. **Preview**
   - See real-time preview of stamp

4. **Apply to Document**
   - Export stamp configuration
   - Use in other applications

## Seal Specifications

### Company Seal
- Circular design with curved text
- Top Text: "For Mcharv Techlabs Pvt. Ltd." (curved along the circle)
- Center: "Shwita" (handwritten style signature)
- Bottom: "Director"
- Color: Blue (#1e40af)
- Border Style: Double circular borders
- Default Size: 120px × 120px

### Director Seal
- Circular design with multiple circles
- Handwritten signature: "Shwita"
- Title: "Director"
- Color: Blue (#1e40af)
- Border Style: Multi-ring circular design
- Default Size: 120px × 120px
